Jenis File di Autodesk Revit


Revit memiliki beberapa jenis file dengan fungsi berbeda yang digunakan dalam proses pemodelan, dokumentasi, dan kolaborasi. Berikut adalah penjelasan masing-masing jenis file:

1.RVT File (Revit Project File)

Fungsi: File utama proyek di Revit.

  • Isi: Semua data model 3D, gambar kerja (sheet), informasi material, parameter proyek, dan lainnya.

  • Kegunaan:

    • Membuat dan menyimpan proyek.

    • Kolaborasi lintas disiplin (Arsitektur, Struktur, MEP).

    • Dibuka dan diedit langsung di Revit.

  • Contoh: GedungKantor.rvt

2. RFA File (Revit Family File)

  • Fungsi: File untuk menyimpan families, yaitu komponen individual yang digunakan dalam model Revit.

  • Isi: Parametric components seperti pintu, jendela, furnitur, alat MEP, dan lainnya.

  • Kegunaan:

    • Menambahkan elemen khusus ke proyek (RVT).

    • Membuat elemen yang bisa digunakan kembali di proyek lain.

  • Jenis Families:

    • System Families: Bawaan Revit (seperti dinding dan lantai).

    • Loadable Families: Bisa diimpor dan disesuaikan (berupa file .rfa terpisah).

    • In-Place Families: Dibuat khusus di dalam proyek tertentu.

  • Contoh: PintuKaca.rfa

3. RTE File (Revit Template File)

  • Fungsi: Template dasar untuk memulai proyek baru di Revit.

  • Isi: Pengaturan awal seperti View (floor plan, section, elevation), standar gambar (title block, sheet), serta parameter dan material khusus.

  • Kegunaan:

    • Mempercepat pembuatan proyek baru dengan standar yang konsisten.

    • Menjaga keseragaman pengaturan di banyak proyek.

  • Contoh: Architectural Template.rte, Structural Template.rte, MEP Template.rte.

4. RFT File (Revit Family Template File)

  • Fungsi: Template dasar untuk membuat file RFA (Family) baru.

  • Isi: Struktur dan parameter awal untuk membuat komponen kustom.

  • Kegunaan: Membuat komponen baru (pintu khusus, furnitur custom) yang dapat digunakan di banyak proyek.

  • Contoh: Metric Door.rft, Generic Model.rft.

5. IFC File (Industry Foundation Classes)

  • Fungsi: Format file open-source untuk berbagi data BIM antar software berbeda.

  • Isi: Informasi model 3D dan metadata terkait.

  • Kegunaan: Kolaborasi lintas platform (Revit, ArchiCAD, Tekla, dsb.) tanpa kehilangan data esensial.

  • Contoh: GedungKantor.ifc

6. DWG File (Drawing Exchange Format)

  • Fungsi: Format gambar 2D/3D dari AutoCAD yang dapat diimpor ke Revit.

  • Isi: Gambar teknis atau referensi CAD.

  • Kegunaan: Mengimpor referensi CAD ke Revit atau menyimpan output Revit agar bisa dibaca software lain.

  • Contoh: DenahLantai.dwg

7. NWC / NWD File (Navisworks File)

  • Fungsi: Format file untuk koordinasi model di Autodesk Navisworks.

  • Isi: Model federasi dari berbagai disiplin yang di-review bersama.

  • Kegunaan: Clash Detection (deteksi benturan) dan simulasi konstruksi 4D (waktu).

  • Contoh: ProyekGedung.nwc

Cara Mendapatkan File di Revit

Revit menyediakan beberapa cara untuk mendapatkan file, tergantung pada kebutuhan dan sumber file yang Anda inginkan. Berikut adalah beberapa metode umum:

1. RVT File (Revit Project File)

  • Membuat sendiri dengan Revit:

    • File → New → Project

  • Buka file project yang sudah ada dari klien atau rekan kerja.

  • Unduhan sampel/contoh proyek dari Autodesk (tersedia di instalasi default Revit)

    • Lokasi: C:\Program Files\Autodesk\Revit 202X\Samples

2. RFA File (Revit Family File)

  • Load Families dari Revit:

    • Insert → Load Family (memuat komponen bawaan)

  • Koleksi Family di Folder Default Revit

    • Lokasi: C:\ProgramData\Autodesk\RVT 202X\Libraries

  • Download dari Internet:

    • Autodesk Content Libraries (mengakses dan mengunduhnya melalui website Autodesk)

      • Untuk versi tahun 2025 dapat menggunakan instalasi file berikut, setelah diunduh lakukan instalasinya dan koleksi family akan muncul pada folder default revit.

    • Situs Pihak Ketiga:

      • BIMObject: https://www.bimobject.com

      • RevitCity: https://www.revitcity.com

      • BIMSmith: https://www.bimsmith.com

      • Revit Files (Dahar Engineer): https://daharengineer.com/community/revit-files

3. RTE File (Revit Template File)

  • Template Default di Revit:

    • Tersedia saat membuat proyek baru: File → New → Project

    • Lokasi: C:\ProgramData\Autodesk\RVT 202X\Templates

  • Custom Template:

    • Bisa membuat sendiri dari proyek yang sudah ada: File → Save As → Template

    • Perusahaan besar biasanya memiliki template yang sudah dikustomisasi untuk kebutuhan mereka.

4. RFT File (Revit Family Template File)

  • File Bawaan Revit:

    • Tersedia saat membuat family baru: File → New → Family

    • Lokasi: C:\ProgramData\Autodesk\RVT 202X\Family Templates

5. IFC File (Industry Foundation Classes)

  • Ekspor dari Revit:

    • File → Export → IFC untuk membuat file IFC dari proyek RVT.

  • Import dari IFC:

    • File → Open → IFC untuk membuka model dari software BIM lain (ArchiCAD, Tekla, dsb.).

  • Situs Pihak Ketiga:

    • BuildingSMART: https://buildingsmart.org

6. DWG File (Drawing Exchange Format)

  • Export dari Revit:

    • File → Export → CAD Format → DWG untuk menyimpan gambar sebagai AutoCAD.

  • Import ke Revit:

    • Insert → Import CAD untuk membuka gambar AutoCAD ke Revit.

  • Dapatkan dari client/rekan kerja/pihak ketiga lainnya

7. NWC / NWD File (Navisworks File)

  • Ekspor dari Revit:

    • File → Export → Navisworks untuk menyimpan model sebagai Navisworks.

  • Import ke Revit: menggunakan plug-in atau koordinasi external.

  • Download File Clash Detection: jika bekerja dalam tim BIM yang menggunakan Navisworks.
